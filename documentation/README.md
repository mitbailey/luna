# General Purpose User-Usable Memory

### __ATmega328__  
(Non-Volatile)
- Flash Program Memory: 32 KB  
- EEPROM: 1 KB  
- SRAM: 2 KB   

### __MLX90393__    
(Non-Volatile, 64 addresses w/ 16 bit words)  
- Customer: 500 bits (0x00 - 0x1F)  
- Melexis: 500 bits (0x20 - 0x3F)  

(Volatile, 64 addresses w/ 16 bit words)   
- Customer: 500 bits (0x00 - 0x1F)  
- Melexis: 500 bits (0x20 - 0x3F)
